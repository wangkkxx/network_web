THE NETWORK TOPOLOGY

方案特点

PROGRAM FEATURES

  • 同台文件服务器可以在同一数据中心或也可以在不同数据中心
  • 同步的文件类型没有限制
  • 可以做到第一次完全同步后，后期做到增量同步，以减少对带宽的消耗
  • 所同步的文件大小和服务器的带宽决定了同步完成锁需要的时间
  • 两台服务器在同一数据中心可以配置内网直连，不同数据中心则都配置公网IP

方案优势

PLAN ADVANTAGE

  • 保持文件同步，防止单台服务器由于硬盘损毁等故障造成数据丢失
  • 对服务器的操作系统没有限制，可以在任何一台机器上灵活部署
  • 文件同步双向性，更加保证了数据的完整性与安全性
  • 方案灵活方便，适合各种网络，投资相对较少

使用群体

USING GROUP

  • 个人博客，网站--在网络上发布个人信息及相关内容展示自己的网络平台。
  • 企业门户网站--提供企业的文化，企业的残品供求信息企业的成功案例，产品的售后服务等等。
  • 电子商务网站--买卖双方实现消费的网上购物、商户之间的网上交易和在线电子支付
